# 薪资计算器

一个基于 2019 年个人所得税法的薪资计算工具，采用累计预扣法计算个人所得税，支持年终奖单独计税和合并计税两种方式，帮助用户准确计算到手薪资。

## 主要功能

### 核心计算功能

- **累计预扣法计算**：按照 2019 年个人所得税法，采用累计预扣法计算每月个人所得税
- **年终奖计税**：支持年终奖单独计税和合并计税两种方式，自动推荐最优方案
- **五险一金计算**：支持自定义五险一金个人缴纳比例（公积金、养老、医疗、失业）
- **专项附加扣除**：支持专项附加扣除项配置

### 数据展示功能

- **年度汇总卡片**：展示全年到手总收入、缴纳个税、五险一金等关键数据
- **每月明细表格**：提供简要版和详细版两种视图，展示每月详细计算过程
- **收入分布图表**：饼图展示全年收入去向分布（到手收入、个税、五险一金）
- **税前总收入统计**：突出显示税前总收入统计基数

### 用户体验优化

- **智能布局**：初始状态输入框居中显示，计算后自动切换为左右分栏布局
- **占位符提示**：输入框提供灰色占位符，提示用户输入格式
- **默认值预设**：五险一金比例、专项附加扣除、发薪月数等提供合理默认值
- **毛玻璃效果**：现代化的 UI 设计，采用毛玻璃效果提升视觉体验
- **参考数据**：提供个人所得税预扣率表和五险一金缴纳比例参考

## 如何运行

### 1. 克隆项目

```bash
git clone https://github.com/Ab4ndon/SalaryCaculator.git
cd SalaryCaculator
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

### 4. 访问应用

打开浏览器访问 `http://localhost:5173`（Vite 默认端口）

### 5. 构建生产版本

```bash
npm run build
```

构建完成后，文件将输出到 `dist` 目录。

### 6. 预览生产构建

```bash
npm run preview
```

## 使用说明

### 基本使用流程

1. **输入月薪**：在"月薪金额"输入框中输入您的月薪（单位：元）
2. **调整参数**（可选）：
   - 修改发薪月数（默认 12 个月）
   - 调整年终奖月数（默认 3 个月）
   - 修改五险一金比例（默认值已预设）
   - 调整专项附加扣除金额（默认 1500 元/月）
   - 选择年终奖计税方式（单独计税或合并计税）
3. **查看结果**：输入月薪后，系统自动计算并显示：
   - 全年到手总收入
   - 全年缴纳个税
   - 个人五险一金
   - 每月明细表格
   - 收入分布图表

### 功能说明

#### 五险一金比例

点击"五险一金个人比例"旁的问号图标，可查看各险种的参考比例范围：
- 住房公积金：5% - 12%
- 养老保险：8%
- 医疗保险：2%
- 失业保险：0.5% - 1%

#### 专项附加扣除

点击"专项附加扣除/月"旁的问号图标，可查看 7 项专项附加扣除项目：
- 3岁以下婴幼儿照护：2000 元/月
- 子女教育：2000 元/月
- 赡养老人：3000 元/月
- 住房租金：1500/1100/800 元/月
- 住房贷款利息：1000 元/月
- 继续教育：400 元/月 或 3600 元/年
- 大病医疗：限额内据实扣除

#### 年终奖计税方式

- **单独计税**：年终奖单独计算个人所得税
- **合并计税**：年终奖与工资合并计算个人所得税

系统会自动比较两种方式，推荐更优方案。

#### 参考税率表

点击右上角"参考税率"按钮，可查看个人所得税预扣率表（2019版），包含 7 个税率档位。

## 计算说明

### 累计预扣法

采用累计预扣法计算个人所得税，随着累计收入增加，适用税率可能逐月提高（跳档）。每月应纳税额计算公式：

```
当月应纳税额 = (累计应纳税所得额 × 适用税率 - 速算扣除数) - 累计已预扣税额
```

### 年终奖计税

- **单独计税**：年终奖除以 12 确定适用税率，单独计算税额
- **合并计税**：年终奖与全年工资合并，按累计预扣法计算
